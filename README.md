PlePer API
===============

A basic PHP wrapper to work with PlePer API.

Installation
------------

It's easiest to install and use the library via [Composer](https://getcomposer.org/).

Step 1: Add to the requires section of your composer.json file.

```
"require": {
    "pleper/api": "0.*"
}
```

Step 2: Install

```
composer install
```
